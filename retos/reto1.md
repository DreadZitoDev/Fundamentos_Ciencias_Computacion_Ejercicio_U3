# Reto 1 - Primer cambio simple

## Objetivo

Practicar `git status`, `git add`, `git commit` y `git log --oneline`.

## Consigna

1. Clonar el repositorio.
2. Completar **solo una línea** en `equipo/integrantes.txt` con nombre, rol e interés.
3. Revisar estado con `git status`.
4. Preparar el archivo para commit con `git add`.
5. Crear un commit con mensaje claro (sugerido: `Reto 1: agrega integrante`).
6. Verificar el resultado con `git log --oneline`.

## Pista

Usar la sintaxis vista en `docs/comandos.md`, pero escribiendo ustedes mismos cada comando.

## Criterio de logro

El historial debe mostrar un commit nuevo y el archivo actualizado correctamente.
